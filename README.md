📊 Customer Behaviour Analysis
📌 Project Overview

This project analyzes customer shopping behavior using transactional data from over 5,000 purchases across multiple product categories. The objective is to uncover insights into customer spending patterns, purchasing trends, product preferences, and subscription behavior.

These insights help in making data-driven business decisions, improving customer retention, and optimizing revenue strategies.

🎯 Objectives
Analyze customer purchasing behavior
Identify high-revenue categories and products
Evaluate the impact of discounts on sales
Segment customers based on purchase history
Understand subscription and loyalty behavior
🗂️ Dataset Description

The dataset includes customer transaction details such as:

  > Customer ID
  > Age & Gender
  > Item Purchased & Category
  > Purchase Amount
  > Review Rating
  > Payment Method
  > Shipping Type
  > Subscription Status
  > Previous Purchases
  > Location
🛠️ Tools & Technologies
